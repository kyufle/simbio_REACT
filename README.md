# Simbio

Simbio és una aplicació web pensada per facilitar la connexió entre usuaris a través de xats privats i la descoberta de perfils afins. La plataforma està orientada a crear una comunitat on els usuaris puguin interactuar, compartir interessos i establir noves col·laboracions o amistats.

## Característiques principals

- **Descobreix Usuaris:** Explora i troba perfils d'altres usuaris, aplicant filtres segons els teus interessos.
- **Missatgeria Privada:** Inicia converses privades i mantén el contacte de manera segura dins de la plataforma.
- **Gestió de Perfil:** Personalitza el teu perfil, incloent-hi foto, interessos i altres dades personals.
- **Notificacions i Alertes:** Rep notificacions de nous missatges i activitats destacades (sistema de missatges flash).
- **Administració:** Panell d'administració per gestionar usuaris i activitats de la plataforma (només per a administradors).

## Tecnologies utilitzades

- **Backend:** PHP
- **Frontend:** HTML, CSS, JavaScript
- **Base de dades:** MySQL
- **Autenticació:** Sistema propi amb gestió de sessions en PHP
